# AI Vehicle Service Backend

Backend for the **AI-Enabled On-Demand Vehicle Service & Breakdown Assistance App**.  
Built with **Node.js, Express, MongoDB**, and optional **Firebase** for push notifications.

---

## 🔹 Features

- **User Roles**: Customer, Garage (Agent), Mechanic, Admin
- **Authentication**: JWT-based with role & admin verification
- **Job Management**: Customers can request service; assign garages and mechanics
- **Payment Handling**: Supports multiple payment methods and tracks status
- **Earnings**: Tracks mechanic/garage earnings automatically
- **Notifications**: In-app notifications stored in MongoDB, optional Firebase push
- **Admin Verification**: Admin verifies garages and mechanics
- **Security**: Helmet, Express Rate Limiter, CORS, dotenv
- **Error Handling**: Global error middleware with professional responses
